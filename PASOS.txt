COMANDOS INICIO PROYECTO NESTJS

1- npm i -g @nestjs/cli (GENERADOR DE PLANTILLA)
2- nest new project (GENERADOR CARPETA DE PROYECTO)
3- cd project (VAMOS A LA CARPETA DEL PROYECTO)
4- npm run start (HACEMOS CORRER NEST)
5- nest g mo dogs (CREAMOS EL MODULO DOGS)
6- nest g co dogs (CREAMOS EL CONTROLADOR)
